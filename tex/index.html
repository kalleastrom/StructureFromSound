Structure from Sound Database (sfsdb)

Here is a database of multi-channel recordings to be used for 'structure from sound' experiments. Each experiment consists of a number of audio files, typically 8, one for each microphone. In each experiment the microphones are stationary, but in unknown position.

  1. Download the matlab code from https://github.com/kalleastrom/StructureFromSound
  2. The folder structure should be:
    • StructureFromSound
      • matlab
        • main.m
        • other matlab code and folders
      • data
        • savefiles (this folder is used to save intermediate results)
        • sfsgt (this folder contains ground truth for experiments)
        • sfsdb (this folder should contain the experiments you want to try. However, this folder is empty when you download it from github)
          • bassh1 (one experiment folder containing 8 sound files)
            • bassh1-1.aiff
            • bassh1-2.aiff
            • ...
            • bassh1-8.aiff
          • bassh2 (another experiment folder containing 8 sound files)
          • bassh3
          • ...
      • tex
        • StructureFromSound.tex (tex file explaining some of the math and algorithms)
  3. Download the raw sound files from this site http://vision.maths.lth.se/sfsdb/ and put the each experiment folder in the folder sfsdb so that you get the right file structure.
  4. Run main.m
